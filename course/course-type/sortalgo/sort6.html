Heap Sort

Heap Sort is a comparison-based sorting algorithm that utilizes a binary heap data structure to efficiently sort elements. It offers a time complexity of O(n log n) in all cases, making it a reliable choice for various sorting applications. The algorithm proceeds in two main phases:

Building a Max-Heap:
The initial array is transformed into a max-heap. In a max-heap, the value of each parent node is greater than or equal to the values of its child nodes. This ensures that the largest element is always at the root of the heap. This phase typically involves a bottom-up approach, starting from the last non-leaf node and "heapifying" upwards.

Sorting Phase:
First, the largest element (root of the max-heap) is swapped with the last element of the heap. Next, the size of the heap is reduced by one, effectively excluding the now-sorted largest element from further heap operations. Then, the heap property is restored by "heapifying" the new root element downwards. This process ensures that the largest remaining element is again at the root. Finally, these steps are repeated until the heap size becomes one, at which point the entire array is sorted in ascending order.
